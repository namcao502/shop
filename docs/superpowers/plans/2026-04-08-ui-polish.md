# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply a unified "clean & modern" design refresh across the shop home page and admin dashboard — shared gradient background, glass panels, amber accent tokens, and consistent typography.

**Architecture:** All changes are purely presentational Tailwind class swaps and one CSS rule addition. No new components, no data layer changes. The gradient is set on `body` in `globals.css` and shows through all pages; glass panels (`bg-white/70 backdrop-blur-md`) float over it. KPICards gains an optional `accent` prop to drive per-card top border colour.

**Tech Stack:** Next.js 14 App Router, Tailwind CSS v4, TypeScript. No test framework — use `npm run build` to catch type errors, then verify visually in the browser (`npm run dev`).

- [ ] **Step 1: Add `accent` prop to KPICards**

Replace the entire contents of `src/components/admin/KPICards.tsx` with:

```tsx
type KPIAccent = "primary" | "warning" | "neutral";

interface KPI {
  label: string;
  value: string;
  sub?: string;
  accent?: KPIAccent;
}

interface KPICardsProps {
  kpis: KPI[];
}

const accentBorder: Record<KPIAccent, string> = {
  primary: "border-t-2 border-amber-500",
  warning: "border-t-2 border-amber-300",
  neutral: "border-t-2 border-stone-200",
};

const valueColour: Record<KPIAccent, string> = {
  primary: "text-stone-900",
  warning: "text-amber-600",
  neutral: "text-stone-900",
};

export function KPICards({ kpis }: KPICardsProps) {
  return (
    <div className="grid grid-cols-2 gap-4 lg:grid-cols-4">
      {kpis.map((kpi) => {
        const accent: KPIAccent = kpi.accent ?? "neutral";
        return (
          <div
            key={kpi.label}
            className={`rounded-xl bg-white p-4 shadow-sm ${accentBorder[accent]}`}
          >
            <p className="text-xs font-semibold uppercase tracking-widest text-stone-400">
              {kpi.label}
            </p>
            <p className={`mt-1 text-2xl font-extrabold tracking-tight ${valueColour[accent]}`}>
              {kpi.value}
            </p>
            {kpi.sub && (
              <p className="mt-0.5 text-xs text-stone-400">{kpi.sub}</p>
            )}
          </div>
        );
      })}
    </div>
  );
}
```
